# Introduction

## Overview

Code deobfuscation is a critical skill for code analysis and reverse engineering. During security exercises, deobfuscated code reveals hidden functionalities—especially in malware that uses obfuscated JavaScript to conceal payload delivery mechanisms.

## Learning Path

This module covers:

- **Locating JavaScript code** - Finding scripts within HTML pages
- **Code Obfuscation Basics** - Understanding obfuscation techniques and use cases
- **Deobfuscation Methods** - Techniques to simplify and decode obfuscated code
- **Message Decoding** - Extracting encoded content
- **Code Analysis** - Understanding functionality and intent
- **HTTP Requests** - Sending basic web requests

By completing this module, you'll be able to analyze obfuscated code, decode its functionality, and understand its behavior.
